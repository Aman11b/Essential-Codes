# What UXD is
- User experience simpley refers to the way product behave and is used in real world.A positiive user experience is one in which the goals of both the user and the organization that created the product is met
- Usability is on attribute of a successful user experience,but usability alone does not make an experience positive for user

### UXD in the sum of a simple equation

business goals + user/customer goals + user interface + back end process + = user experience design

### UXD in the sum of these attribites
 useful + desireable + accessible + credible + findable + usable = valuable (UX honeycomb)

### UXD is the result of multiple activities

user research + interaction design + visual design + infromation architecture + font end development + writing/content + user testing 

> UXD means problem solving

>  UXD explores feasible solution ot design problems
- it uncovers user needs and business goals(and area where the two overlap)
- it ensures products meets or exceed user expectationb,increase adoption use and loyality while reducing support

### UXD ensures peopel use things
because we use things when:
- we can easily figure out how to accomplish it
